OMDL Community

How to get involved

Joining the OMDL group

Anyone can join the OMDL Google Group, which is where the specification is discussed and developed.

Contributing to the specification

Anyone is welcome to make contributions to the specification using the Google Group (above). However, for contributions to make it into a released spec you need to follow the licensing instructions. This is to ensure that anyone is free to make use of the specification without worrying about copyright or patent issues.

Implementation

Implementations are very, very welcome! If you have any work to share, post to the group (above) and we'll link to it from this site.

Reporting bugs and issues

Report any bugs or issues on the OMDL group list above.
